# ✅ Exercícios Desenvolvidos

## Exercício 1

Programa que imprime:

```text
1
2 2
3 3 3
...
```

Utilizando função com parâmetro inteiro.

---

## Exercício 2

Programa que imprime:

```text
1
1 2
1 2 3
...
```

Utilizando função com parâmetro inteiro.

---

## Exercício 3

Função que recebe três números e retorna a soma deles.

---

## Exercício 4

Função que retorna:

* 'P' → número positivo
* 'N' → número negativo ou zero

---

## Exercício 5

Programa que calcula o valor de um produto com imposto utilizando a função:

```text
somaImposto(taxaImposto, custo)
```

---

## Exercício 6

Programa que converte horário:

* Formato 24h → 12h
* Exemplo:

```text
14:25 → 2:25 P.M.
```

Com repetição para novas conversões.
